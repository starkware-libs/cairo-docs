# minmax

Takes two comparable values `a` and `b` and returns
a tuple with the smaller value and the greater value.
# Examples

```cairo
use core::cmp::minmax;

assert!(minmax(0, 1) == (0, 1));
assert!(minmax(1, 0) == (0, 1));
```
