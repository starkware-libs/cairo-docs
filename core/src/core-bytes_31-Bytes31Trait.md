## Trait functions

### at

Returns the byte at the given index (LSB's index is 0).
Assumes that `index < BYTES_IN_BYTES31`. If the assumption is not met, the behavior is
undefined.
# Examples

```cairo
let bytes: bytes31 = 1_u8.into();
assert!(bytes.at(0) == 1);
```
