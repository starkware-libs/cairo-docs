# u256_inv_mod

Returns the inverse of `a` modulo `n`, or `None` if `a` is not invertible modulo `n`.
All `a`s will be considered not invertible for `n == 1`.
# Examples

```cairo
use core::math::u256_inv_mod;

let inv = u256_inv_mod(3, 17);
assert!(inv == Some(6));
```
